# 💰 Expense Tracker (Java + JDBC + Maven + PostgreSQL)

A simple backend Java application to manage personal expenses using JDBC and PostgreSQL.  
This project demonstrates core backend concepts like database connectivity, DAO pattern, and Maven project structure.

---

## 🚀 Features

- Add new expenses
- View all expenses
- PostgreSQL database integration
- Clean DAO-based architecture
- Maven build system

---

## 🛠️ Tech Stack

- Java 17+
- Maven
- JDBC
- PostgreSQL
- IntelliJ IDEA
